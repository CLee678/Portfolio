Graphical User Interface of Green Equity Calculator

The Green Equity Calculator is a collaborative project aimed at generating a comprehensive green equity index for various census tracts within a state, utilizing tree canopy and census data. My role in this project involved designing the graphical user interface (GUI), which allows users to input geographical data, such as state and county FIPS codes, and upload relevant files. The application generates tailored plant recommendations, shapefiles, and a choropleth map that visualizes the green equity index score across different counties.
Geospatial Analysis Tools/Platforms: Python, Tkinter, pandas, GeoPandas, NumPy, Rasterio, Matplotlib
Data Sources: Census Data API, NLCD
Data Types: Raster, Shapefile

Geospatial Prediction of Fire Radiative Power (FRP) in California: Insights from Machine Learning Models

This project aimed to predict wildfire occurrence and severity in California using key environmental variables such as FRP, vegetation, topography, and climate data. After conducting Exploratory Data Analysis and Principal Component Analysis, six significant predictors were selected: precipitation, mean temperature, maximum vapor pressure deficit, elevation, slope, and tree cover. Three regression models—Random Forest, XGBoost, and Neural Networks—were trained and evaluated. Among them, Random Forest performed best, achieving the lowest RMSE (35.0) and the highest R² (0.0708). The map illustrates FRP predictions across California, with yellow indicating low and red indicating high wildfire intensity.
Geospatial Analysis Tools/Platforms: RStudio
Data Sources: 2020 VIIRS from FIRMS, NLCD 2020 USFS CONUS from MRLC, 2019 Global Forest Canopy Height from GLAD, Geodata, Prism, Tigris
Data Types: Raster, Vector, CSV, GeoTIFF, Shapefile

Remote Sensing Analysis of California Wildfires in 2020 Using Google Earth Engine

The analysis utilizes the MODIS MCD64A1 image collection for data on burned regions. A burned area mask is generated by filtering these images for the year 2020. On the resulting map, the red polygon shows the boundary of California, while the black regions indicate the areas burned in 2020. The analysis estimates that wildfires in California burned approximately 3,764,985 acres in 2020.
Geospatial Analysis Tools/Platforms: Google Earth Engine, JavaScript
Data Sources: MODIS(MCD64A1), California Open Data
Data Types: Raster, Shapefile

Location Analysis for a New Healthy Food Store in Philadelphia

The map identifies candidate locations for a new healthy food store in Philadelphia. These zones are situated within the Philadelphia Empowerment Zones and are accessible within 2,000 feet of a SEPTA High Speed Station or Regional Rail Station. They are also located at least 1,200 feet away from existing fresh produce vendors, including Farmers Markets and stores in the Healthy Corner Stores Initiative. The largest candidate zone covers an area of 4,049,470 square feet.
Map Type: Static Map
Geospatial Analysis Tools/Platforms: Python, ArcPy, ArcGIS Pro
